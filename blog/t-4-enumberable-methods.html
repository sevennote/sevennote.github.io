Enumberable Methods Group-by

3/22/2015

Group by is an enumberable method that can take elements from a certain hash, organize and display them by certain properties. A good example would be.

list = { "apple" => 1, "banana" => 2, "pear" => 3 } puts list.group_by {|k,v| k == "apple"}

The result will display. {true=>[["apple", 1]], false=>[["banana", 2], ["pear", 3]]}
