class Fun def initialize(beach,food) @beach = beach @food = food end def eat_food p "#{@food} nom nom nom" end def surf's up p "#{@beach} surf is up at this beach" end n

Classes are used to develop properties for objects. The instance variables carry over the properties of the initialize method into the eat_food and surf's up method. Instance variables allow for data to be shared between classes.
